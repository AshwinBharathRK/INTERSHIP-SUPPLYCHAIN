# 📖 Overview

Modern supply chains are complex, dynamic, and vulnerable to disruptions such as supplier delays, fluctuating demand, inventory shortages, and transportation bottlenecks.

This platform transforms traditional supply chain management into an intelligent decision-support system by combining:

* 🤖 Artificial Intelligence
* 📊 Business Analytics
* 🌍 Interactive Supply Chain Visualization
* 🏭 Digital Twin Simulation
* 📦 Inventory Optimization
* 🚚 Logistics Intelligence
* 📈 Demand Forecasting

Rather than displaying static dashboards, the application provides an immersive experience where users can analyze, simulate, and optimize supply chain operations using real-world data.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/AshwinBharathRK/INTERSHIP-SUPPLYCHAIN.git
cd INTERSHIP-SUPPLYCHAIN
```

## Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate

pip install -r requirements.txt

uvicorn main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
